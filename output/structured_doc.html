model.py
This script uses the OpenAI API to summarize the code in a specified GitHub repository. It takes in a language model and a dictionary of files and code as arguments. It then iterates through the files and code, skipping the __init__ file, and uses the OpenAI API to generate a summary of the code for each file. The summaries are stored in a dictionary and returned.
